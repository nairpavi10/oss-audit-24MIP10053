## Scripts Description

### Script 1: System Identity Report
This script displays system information including Linux distribution name, kernel version, current user, home directory, system uptime, and current date/time. It also prints a message about the open-source license.


### Script 2: User Input and Calculation
This script takes input from the user and performs basic calculations. It demonstrates how user input can be handled in shell scripting.


### Script 3: Loop Example
This script prints numbers from 1 to 5 using a loop. It demonstrates the use of loops in shell scripting.


### Script 4: Even or Odd Checker
This script takes a number as input and checks whether it is even or odd using conditional statements.


### Script 5: Manifesto Script
This script asks questions and generates a simple text file (manifesto.txt) based on user responses. It demonstrates file creation and writing.
